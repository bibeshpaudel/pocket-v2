Status pill in Pocket's earth tones; for validation results, counts, and tool states.

```jsx
<Badge kind="ok" dot>Valid JSON</Badge>
<Badge kind="danger">Parse error</Badge>
<Badge kind="accent">New</Badge>
<Badge kind="neutral">2,148 chars</Badge>
```

ok=moss, danger=clay, warn=ochre, accent=amber. Never use it as a button.
