Inline Lucide icon that inherits currentColor; use for ALL pictography (no emoji, no hand-drawn SVG).

```jsx
<Icon name="braces" size={18} />
```

Sizes: 16 inline/controls, 18 tool-card glyphs, 20 nav. Names are kebab-case from lucide.dev. Fetched from CDN once and cached; renders empty until loaded.
